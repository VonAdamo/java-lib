# Git Demo Project

this is a demonstration using **Java 25**

## Project Overview

- **Build System:** Maven
- **Java Version:** 25
- ** Main Class:** `com.example.Main`

## Features

- Uses modern Java 252 features, including instance main methods and implicitly declared local variables.
- Minimalist project structure for demonstration purposes.

## How to Run

To compile and run the project, use:

```bash
mvn compile exec:java -D "exec.mainClass=com.example.Main"
```
